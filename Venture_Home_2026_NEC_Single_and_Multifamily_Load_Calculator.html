Calculation Structure

• General load, cooking, dryer, and qualifying fastened appliances form the dwelling demand-factor bucket.
• The first 8,000 VA is taken at 100%; the remainder is taken at 40%.
• EVSE is excluded from that bucket and added afterward at the selected 100% or 80%.
• HVAC is excluded from that bucket and added afterward based on the selected HVAC treatment.
• Additional load rows are added afterward, with an optional 125% multiplier.

Multifamily Project Info 2+ dwelling units

Dwelling Unit Setup

Two-unit branch: each dwelling is calculated independently using the one-family optional workflow, then combined. Three or more units: the combined dwelling load uses the multifamily demand table or manual override. House/common loads are added afterward.
